# Taskhlant

A simple task management application.

## Description

Taskhlant is a lightweight task management tool designed to help you organize and track your tasks efficiently.

## Features

- Task creation and management
- Simple and intuitive interface
- Easy to use and extend

## Installation

```bash
# Clone the repository
git clone https://github.com/xmhhhh82/taskhlant.git
cd taskhlant

# Install dependencies (if any)
pip install -r requirements.txt
```

## Usage

```bash
# Run the application
python main.py
```
